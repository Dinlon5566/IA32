fms - 浮点乘法减法

fms

操作

格式

(qp) fms.pc.sf f1 = f3, f4, f2 F1

说明

FR f3 与 FR f4 的乘积计算到无限精度，然后减去 FR f2，同样精确到无限精度。随后按照 FPSR.sf.rc 指定的取整模式，将结果值取整到 pc（及可能的 FPSR.sf.pc 与 FPSR.sf.wre）指定的精度。取整后的结果放入 FR f1。

如果 FR f3、FR f4 或 FR f2 中的任何一个是 NaTVal，则将 NaTVal 放入 FR f1，而不放入计算的结果。

如果 f2 是 f0，则执行 IEEE 乘法运算，而不是乘法与减法。请参阅 fmpy - 浮点乘法。

FP 异常

  • 无效操作 (V) 下溢 (U)

  • 非规格化/非常规操作数 (D) 上溢 (O)

  • 软件协助 (SWA) 错误：不精确 (I)

  • 软件协助 (SWA) 陷阱

  • 中断：“非法操作”错误与“浮点异常”错误

  • 禁用“浮点寄存器”错误与“浮点异常”陷阱
