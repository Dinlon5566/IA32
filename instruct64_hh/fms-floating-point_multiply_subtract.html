fms - 浮點乘法減法

fms

操作

格式

(qp) fms.pc.sf f1 = f3, f4, f2 F1

說明

FR f3 與 FR f4 的乘積計算到無限精度，然後減去 FR f2，同樣精確到無限精度。隨後按照 FPSR.sf.rc 指定的取整模式，將結果值取整到 pc（及可能的 FPSR.sf.pc 與 FPSR.sf.wre）指定的精度。取整后的結果放入 FR f1。

如果 FR f3、FR f4 或 FR f2 中的任何一個是 NaTVal，則將 NaTVal 放入 FR f1，而不放入計算的結果。

如果 f2 是 f0，則執行 IEEE 乘法運算，而不是乘法與減法。請參閱 fmpy - 浮點乘法。

FP 異常

  • 無效操作 (V) 下溢 (U)

  • 非規格化/非常規運算元 (D) 上溢 (O)

  • 軟體協助 (SWA) 錯誤：不精確 (I)

  • 軟體協助 (SWA) 陷阱

  • 中斷：“非法操作”錯誤與“浮點異常”錯誤

  • 禁用“浮點暫存器”錯誤與“浮點異常”陷阱
