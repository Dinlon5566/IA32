loop 分支类型

cloop 分支类型针对简单的计数循环而提供。此分支类型使用“循环计数”应用寄存器 (LC) 确定分支条件，而不使用限定谓词：

如果 LC 寄存器不等于零，则它将递减并执行分支。

除这些简单的分支类型以外，还有四种用于加速模数调度循环的分支类型。其中两个是计数循环（使用 LC 寄存器），另外两个是 while 循环（使用限定谓词）。这些循环类型使用寄存器循环移位提供寄存器重命名功能，并且它们使用断定关闭对应空管道阶段的指令。

“结束计数”应用寄存器 (EC) 用于统计结束阶段；对于某些 while 循环，则是起始阶段的一部分。在结束阶段 EC 每次都会递减，对于大部分的循环，EC 为一时，管道已排放且循环已退出。对于特定类型展开的软件管道优化循环，br.cexit 或 br.wexit 的目标会设置为序列中的下一个指令束。在此种情况下，EC 为一时，管道可能未完全排放，并继续排放到 EC 为零。

对于这些模数调度循环类型，分支是否执行的计算取决于核心分支条件（LC 用于计数型，限定谓词用于 while 型）以及结束条件（EC 是否大于一）。

这些分支类型分为两类：顶置与出口。循环决定位于循环主体的底部时，使用顶置型（ctop 与 wtop），这样，执行分支将继续循环，而不执行的分支则退出循环。循环决定位于循环主题底部之外的某个位置时，使用出口型（cexit 与 wexit），这样不执行的分支将继续循环，而执行的分支则退出循环。出口型也可以用在展开的管道式循环中的中间点上。

模数调度循环类型包括：

  • ctop 与 cexit

  • wtop 与 wexit

循环型分支（CLOOP、CTOP、CEXIT、WTOP 及 WEXIT）仅允许在指令束的指令槽 2 中使用。在指令槽 0 或 1 中执行这类指令将导致“非法操作”错误，无论分支是否执行。
