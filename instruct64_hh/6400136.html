加法指令

add

操作 字体惯例

指令类型 A

格式

(qp)  add  r1 = r2, r3 register_form

(qp)  add  r1 = r2, r3, 1 plus1_form, register_form

(qp)  add  r1 = imm, r3 pseudo-op

(qp)  adds  r1 = imm14, r3 imm14_form

(qp)  addl  r1 = imm22, r3 imm22_form

说明

将两个源操作数（及可选的常数 1）相加，结果放入 GR r1。在寄存器这种形式中，第一个操作数是 GR r2；在 imm_14 这种形式中，第一个操作数取自符号扩展的 imm14 编码字段；在 imm22_form 中，第一个操作数取自符号扩展的 imm22 编码字段。在 imm22_form 中，GR r3 只能指定 GR 0、1、2 以及 3。

plus1_form 只可用于 register_form（尽管通过调整立即数也能在立即数形式中达到同样的效果）。

立即数形式的伪操作会根据立即数操作数大小与 GR r3 中值选择 imm14_form 或 imm22_form。
