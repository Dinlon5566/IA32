LSL(1) (Thumb*) - 逻辑左移

语法

LSL dest, op1, #value

其中：

 dest

目标寄存器

 op1

包含第一个操作数的寄存器

 value

5 位立即数，形成 [0...31] 范围内的移位。

说明

此格式的 LSL 指令按 value 指定的数字左移 op1 中的位，结果存储到 dest。如果 value 包含的移位量在 [1...31] 范围内，则直接编码。如果 value 包含的移位量等于 [0]，则编码成 value ==0。 结果表示将寄存器值乘以 2 的常数次幂。

寻址模式

无操作数。
