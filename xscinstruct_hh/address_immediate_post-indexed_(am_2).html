立即数事后变址 (AM 2)

语法

[basereg], #+/-value

其中：

 basereg

指定包含基址的寄存器。

 value

指定 12 位立即数偏移量，与 basereg 的值一起使用，以形成地址。

说明

寻址模式“立即数事后变址”执行以下操作（伪代码说明）：

addr = basereg
if ConditionPassed (condition) then
if U == 1 then
   basereg = basereg + value
else/* U == 0 */
   basereg = basereg - value

此寻址模式使用基址寄存器 basereg 的值作为内存访问地址。

如果指令中指定的条件与条件代码状态匹配，则立即数偏移量的值会加到基址寄存器 basereg 的值，或从中减去。然后它写回 basereg。

对于带自动更新指针值的数组的指针访问，此寻址模式非常有用。

B 位表示无符号字节 ( B==1 ) 或字访问 ( B==0 )。

L 位表示“加载”( L==1 ) 或“存储”( L==0 ) 指令。

备注

指令 LDRBT、LDRT、STRBT 及 STRT 仅支持事后变址寻址模式。对于这些指令，上述位模式会作如下修改：
bit[21]（W 位）是 1，而不是上述的 0。

对于仅支持事后变址寻址模式（ LDRBT、LDRT、STRBT 或 STRT）的指令，语法 [basereg] 视作 [basereg], #0 的缩写。对于其它所有指令，语法 [basereg] 不得视作 [basereg], #0 的缩写。

注意

如果 R15 指定为寄存器 basereg，则结果将会无法预知。

示例

LDR    R9, [R7], #5
