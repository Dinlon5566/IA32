按寄存器算术右移 (AM1)

语法

op2, ASR op3

其中：

 op2

指定包含要移位的值的寄存器。

 ASR

指示算术右移。

 op3

指定包含移位值的寄存器。

说明

数据处理操作数“按寄存器算术右移”执行以下操作（伪代码说明）：

if op3[7:0] == 0 then
  addr_mode = op2
  shifter_carry_out = C Flag
else if op3[7:0] < 32 then
  addr_mode = op2 Arithmetic_Shift_Right op3[7:0]
  shifter_carry_out = op2[op3[7:0] - 1]
else /* op3[7:0] >= 32 */
  if op2[31] == 0 then
      addr_mode = 0
      shifter_carry_out = op2[31]
  else /*op2[31] == 1 */
      addr_mode = 0xFFFFFFFF
      shifter_carry_out = op2[31]

此数据处理操作数提供算术右移寄存器的有符号值（除以 2 的可变次幂）。

此指令操作数代表算术右移寄存器 op2 的值，移位量等于寄存器 op3 最低有效字节中的值。在腾出的位中，将插入 op2 ( op2 [31] ) 的符号位。最后移出的位是移位器的进位。如果移位量超过 32，则此位是 op2 的符号位。如果移位量等于 0，则进位是 C 标志。
