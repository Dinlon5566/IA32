WAVG2 - 2 像素平均

语法

WAVG2length[R][condition] dest, op1, op2

其中：
length 所需的数据长度，可以由以下字符之一指定：
B -- 操作 8 位（字节）SIMD 数据
H -- 操作 16 位（半字）SIMD 数据
R （可选）
对于使用 +1 的取整，此处放入字符 R。
其它情况则忽略，不进行取整
condition 可选。16 个条件之一。请参阅条件执行
dest 目标，“英特尔(R) 无线 MMX™”技术数据寄存器
op1 第一个源操作数，“英特尔(R) 无线 MMX™”技术数据寄存器
op2 第二个源操作数，“英特尔(R) 无线 MMX™”技术数据寄存器
 

说明

在允许选择是否 +1 取整的 8 或 16 位数据无符号矢量上执行 op1 与 op2 的 2 像素平均计算，并将结果放入“目标”寄存器 dest。
取整指定在除以 2 之前，是否将中间结果加上 1。

指令中指定的任何条件必须与条件代码状态匹配，否则不执行指令。

if R specified then Round = 1, Else Round = 0

if (B Specified) then

dest[byte 7] = (op1[byte 7] + op2[byte 7] + Round)/2

dest[byte 4] = (op1[byte 4] + op2[byte 4] + Round)/2
