按立即数循环右移 (AM 1)

语法

op2, ROR #sh_imm

其中：

 op2

指定包含要循环移位的值的寄存器。

 ROR

表示循环右移。

 sh_imm

指定循环移位量。此值介于 1 与 31 之间。sh_imm == 0 时，执行 RRX 操作（扩展循环右移）。如需有关详细信息，请参阅“数据处理操作数：扩展循环右移 (AM 1)”。

说明

数据处理操作数“按立即数循环右移”执行以下操作（伪代码说明）：

if sh_imm == 0 then
  See Data-Processing Operands: Rotate Right with Extend (AM 1)
else /* sh_imm < 0 */
  addr_mode = op2 Rotate_Right sh_imm
  shifter_carry_out = op2[sh_imm -1]

此数据处理操作数提供按常数循环移位的寄存器值。

此指令操作数代表按立即数循环右移的寄存器 op2 的值，该立即数介于 1 与 31 之间。循环移位到右端以外时，将一些位插入左侧腾出的空位上。最后一个循环移位到右端以外的位是移位器的进位。
