按寄存器逻辑右移 (AM 1)

语法

op2, LSR op3

其中：

 op2

指定包含要移位的值的寄存器。

 LSL

表示逻辑右移。

 op3

指定包含移位值的寄存器。

说明

数据处理操作数“按寄存器逻辑右移”执行以下操作（伪代码说明）：

if op3[7:0] == 0 then
  addr_mode = op2
  shifter_carry_out = C Flag
else if op3[7:0] < 32 then
  addr_mode = op2 Logical_Shift_Right op3[7:0]
  shifter_carry_out = op2[op3[7:0] - 1]
else if op3[7:0] == 32 then
  addr_mode = 0
  shifter_carry_out = op2[31]
else /*op3[7:0] >32 */
  addr_mode = 0
  shifter_carry_out = 0

此数据处理操作数提供右移的（除以二的可变次幂）无符号寄存器值。

此指令操作数代表寄存器 op2 的值，此值按照寄存器 op3 最低有效字节中的值进行过逻辑右移。在腾出的空位中插入零。最后移出的位是移位器的进位。如果移位量大于 32，则此位等于零。如果移位量等于 0，则进位是 C 标志。
