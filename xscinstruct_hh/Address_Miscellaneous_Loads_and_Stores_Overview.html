其它加载与存储概述

“其它加载与存储”寻址模式分成 6 个子模式。它们用于计算加载与存储有符号或无符号半字、加载有符号字节或加载与存储双字指令的地址。此寻址模式的一般语法在后续部分提供。

一般语法

LDR|STR [condition] H|SH|SB|D dest, addr_mode

其中：

 LDR

指定“加载寄存器”指令。

 STR

指定“存储寄存器”指令。

 condition

16 个条件之一。请参阅条件代码状态。

 H

将指令扩展到“加载寄存器半字”( LDRH ) 指令或“存储寄存器半字”( STRH ) 指令。如需有关详细信息，请参阅 LDRH 指令或 STRH 指令。

 SH

将指令扩展到“加载寄存器有符号半字”( LDRSH ) 指令。如需有关详细信息，请参阅 LDRSH 指令。

 SB

将指令扩展到“加载寄存器有符号字节”( LDRSB ) 指令。如需有关详细信息，请参阅 LDRSB 指令。

 D

将指令扩展到 LDRD 指令或 STRD 指令。如需有关详细信息，请参阅 LDRD 指令或 STRD 指令。

 dest

指定目标寄存器。

 

addr_mode 是以下 6 种子模式之一：

子模式

说明

 [basereg, #+/-value]

偏移量选项：
将 8 位立即数偏移量的值加到基址寄存器，或从中减去。请参阅其它加载与存储：立即数偏移量 (AM 3)。

 [basereg, +/-indexreg]

偏移量选项：
将变址寄存器的值加到基址寄存器，或从中减去。请参阅其它加载与存储：寄存器偏移量 (AM 3)。

 [basereg #+/-value]!

事先变址选项：
将 8 位立即数偏移量的值加到基址寄存器，或从中减去。如果传递了某个条件，则将值写回基址寄存器。请参阅其它加载与存储：立即数事先变址 (AM 3)。

 [basereg, +/-indexreg]!

事先变址选项：
将变址寄存器的值加到基址寄存器，或从中减去。如果传递了某个条件，则将值写回基址寄存器。请参阅其它加载与存储：寄存器事先变址 (AM 3)。

 [basereg], #+/-value

事后变址选项：
将 8 位立即数偏移量的值加到基址寄存器，或从中减去。然后将值写回基址寄存器。请参阅其它加载与存储：立即数事后变址 (AM 3)。

 [basereg], +/-indexreg

事后变址选项：
如果传递某个条件，则将变址寄存器的值加到基址寄存器，或从中减去。然后将值写回基址寄存器。请参阅其它加载与存储：寄存器事后变址 (AM 3)。

其中：

 P

P == 0 表示事后变址寻址。基址寄存器值用作内存地址。偏移量应用于基址寄存器的值，并写回基址寄存器。
P == 1 表示偏移量寻址或事先变址寻址。（这由 W 位确定。）内存地址是通过将偏移量应用于基址寄存器值而生成的。

 U

表示是偏移量加到基址还是从中减去：
如果 U == 1，则偏移量加到基址上。
如果 U == 0，则偏移量从基址中减去。

 W

W 的含义取决于 P 的设置：
如果 P == 0，则 W 必须是 0。否则结果将会无法预知。
如果 P == 1：
如果 W == 0，则基址寄存器不更新（偏移量寻址）。
如果 W == 1，则计算的内存地址写回基址寄存器（事先变址寻址）。

 L

表示“加载”或“存储”：
L==1 表示“加载”。
L == 0 表示“存储”。

 S

表示有符号或无符号半字访问：
S == 0 表示无符号半字访问。
S == 1 表示有符号半字访问。

 H

表示半字访问或字节访问：
H == 0 表示字节访问。
H == 1 表示半字访问。

     

无符号字节

如果 S 与 H 表示无符号字节（S == 0 且 H == 0），则指令不使用此寻址模式。这是一个乘法指令，即 SWP 或 SWPB 指令，或是算术或加载/存储指令扩展区中未分配的指令。

无符号字节通过指令 LDRB、LDRBT、STRB 及 STRBT 访问。这些指令使用寻址模式 2。如需有关详细信息，请参阅“寻址模式 2：加载与存储字或无符号字节”。

有符号存储

如果 S 与 L 表示有符号存储指令（S == 1 且 L == 0），则指令是加载/存储扩展区中未分配的指令。有符号字节与半字使用指令 STRB 与 STRH 存储。
