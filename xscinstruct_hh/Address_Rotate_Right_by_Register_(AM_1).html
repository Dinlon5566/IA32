按寄存器循环右移 (AM 1)

语法

op2, ROR op3

其中：

 op2

指定包含要循环移位的值的寄存器。

 ROR

表示循环右移。

 op3

指定包含循环移位值的寄存器。

说明

数据处理操作数“按寄存器循环右移”执行以下操作（伪代码说明）：

if op3[7:0] == 0 then
  addr_mode = op2
  shifter_carry_out = C Flag
else if op3[4:0] == 0 then
  addr_mode = op2
  shifter_carry_out = op2[31]
else /* op3[4:0] > 0 */
  addr_mode = op2 Rotate_Right op3[4:0]
  shifter_carry_out = op2[op3[4:0] - 1]

数据处理操作数提供按变量值循环移位的寄存器值。

此指令操作数代表寄存器 op2 的值，此值按寄存器 op3 最低有效字节中的值进行循环右移。循环移位到右端以外时，一些位将插入左侧腾出的空位上。最后一个循环移位到右端以外的位是移位器的进位。如果移位量等于 0，则进位是 C 标志。
