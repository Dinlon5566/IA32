TINSR - 传输与插入

语法

TINSRlength[Condition] dest, op1, #Imm3

其中：

length

所需的数据长度，可以由以下字符之一指定：

B -- 插入 8 位（字节）操作

H -- 插入 16 位（半字）操作

W -- 插入 32 位（字）操作

condition

可选。16 个条件之一。请参阅：条件执行。

dest

目标，“英特尔(R) 无线 MMX™”技术数据寄存器

op1

源操作数，“英特尔 XScale(R)”核心数据寄存器

Imm3

指定插入哪个字节/半字/字。

 

说明

从“英特尔 XScale(R)”核心源处理器 op1 传输 8-/16-/32 位数据，并将它插入“英特尔(R) 无线 MMX™”技术目标寄存器 dest 中 3 位立即数指定的位置。

指令中指定的任何条件必须与条件代码状态相匹配，否则不执行该指令。

备注

“目标”寄存器中其它的一些字段保持不变。

影响的 SIMD 标志

不影响 SIMD PSR 标志 (wCASF) 或“饱和”标志 (wCSSF)。

操作

if (B Specified) then

dest[Byte #Imm3[2:0]] = op1[7:0]

else if (H Specified)

dest[Half #Imm3[1:0]] = op1[15:0]

else if (W Specified) then

dest[Word #Imm3[0]] = op1[31:0]
