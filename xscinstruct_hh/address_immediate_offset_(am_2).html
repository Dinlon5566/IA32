立即数偏移量 (AM 2)

语法

[basereg, #+/-value]

其中：

 basereg

指定包含基址的寄存器。

 value

指定 12 位立即数偏移量，与 basereg 的值一起使用，以形成地址。

说明

寻址模式“立即数偏移量”执行以下操作（伪代码说明）：

if U == 1 then
  addr = basereg + value
else /* U == 0 */
  addr = basereg - value

此寻址模式将立即数偏移量的值加到基址寄存器 basereg 的值，或从中减去，以计算地址。它用于访问结构字段、访问堆栈帧中的参数与局部变量。如果偏移量是零，则生成的地址是基址寄存器 basereg 的值。

B 位表示无符号字节 ( B == 1 ) 或字访问 ( B == 0 )。

L 位表示“加载”( L == 1 ) 或“存储”( L == 0 ) 指令。

备注

语法 [basereg] 视作 [basereg, #0] 的缩写。如果指令是仅允许使用事后变址寻址模式之一的指令（LDRBT、LDRT、STRBT 或 STRT），则语法 [basereg] 不能 视作 [basereg, #0] 的缩写。

如果 R15 指定为寄存器 basereg，则使用的值是当前指令的地址加 8。

示例

LDR    R3, [R1, #0xFF]
