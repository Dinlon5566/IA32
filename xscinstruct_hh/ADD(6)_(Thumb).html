ADD(6) (Thumb*) - 加法

语法

ADD dest, SP, (#value * 4)

其中：

dest

目标寄存器

SP

SP 相对寻址

value

8 位立即数，代表第二个操作数。[0..1020] 范围内 4 的任何倍数。

说明

此格式的 ADD 指令将四倍 value 与 SP 相加，并将产生的 SP 相对地址存储到 dest。

寻址模式

无操作数。
