WACC - 无符号累加

语法

WACClength[condition] dest, op1

其中：

length
所需的数据长度，可以由以下字符之一指定：
B -- 操作 8 位（字节）SIMD 字段
H -- 操作 16 位（半字）SIMD 字段
W -- 操作 32 位（字）SIMD 字段
condition
可选。16 个条件之一。请参阅：条件执行。
dest
目标，“英特尔(R) 无线 MMX™”技术数据寄存器
op1
源操作数，“英特尔(R) 无线 MMX™”技术数据寄存器

 

说明

在源寄存器 op1 的多个矢量上执行无符号累加，并将结果写入“目标”寄存器 dest。操作可以在字节、半字以及字大小的字段上执行。

指令中指定的任何条件必须与条件代码状态匹配，否则不执行指令。

操作

if (B Specified) then

dest = op1[63:56] + op1[55:48] + op1[47:40] + op1[39:32]
+ op1[31:24] + op1[23:16] + op1[15:8] +op1[7:0]

else if (H Specified)

dest = op1[63:48] + op1[47:32] + op1[31:16] + op1[15:0]

else if (W Specified) then

dest = op1[63:32] + op1[31:0]
