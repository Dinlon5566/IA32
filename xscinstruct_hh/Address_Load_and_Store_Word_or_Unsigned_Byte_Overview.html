載入與儲存字或無符號位元組概述

“載入與儲存字或無符號位元組”定址模式分成 9 個子模式。它們用於計算載入與儲存無符號位元組指令的地址。此定址模式的一般語法在後續部分提供。

一般語法

LDR|STR [condition] [B] [T] dest, addr_mode

其中：

 LDR

指定“載入暫存器”指令。

 STR

指定“儲存暫存器”指令。

 condition

16 個條件之一。請參閱條件程式碼狀態。

 B

將指令擴充套件到“載入暫存器位元組”(LDRB) 指令或“儲存暫存器位元組”( STRB ) 指令。如需有關詳細資訊,請參閱 LDRB 指令或 STRB 指令。

 T

將指令擴充套件到“帶轉譯載入暫存器”(LDRT) 指令，或“帶轉譯儲存暫存器”( STRT ) 指令。如需有關詳細資訊，請參閱 LDRT 指令或 STRT 指令。

 dest

指定目標暫存器。

 

addr_mode 是以下 9 種子模式之一：

子模式

說明

 [ basereg , #+/- value ]

偏移量選項：
將 12 位立即數偏移量的值加到基址暫存器，或從中減去。請參閱載入/儲存字或無符號位元組：立即數偏移量 (AM 2)。

 [ basereg , +/- indexreg ]

偏移量選項：
將變址暫存器的值加到基址暫存器，或從中減去。請參閱載入/儲存字或無符號位元組：暫存器偏移量 (AM 2)。

 [ basereg , +/- indexreg , shift # sh_imm ]

偏移量選項：
將移位的或循環移位的變址暫存器值加到基址暫存器，或從中減去。shift 指定五種移位型別之一。請參閱載入/儲存字或無符號位元組：調整的暫存器偏移量 (AM 2)。

 [ basereg #+/- value ]!

事先變址選項：
將 12 位立即數偏移量的值加到基址暫存器，或從中減去。如果傳遞了某個條件，則將值寫回基址暫存器。請參閱載入/儲存字或無符號位元組：立即數事先變址 (AM 2)。

 [ basereg , +/- indexreg ]!

事先變址選項：
將變址暫存器的值加到基址暫存器，或從中減去。如果傳遞了某個條件，則將值寫回基址暫存器。請參閱載入/儲存文字或未有符號位元組：暫存器預先變址 (AM 2)。

 [ basereg , +/- indexreg , shift # sh_imm ]!

事先變址選項：
將移位的或循環移位的變址暫存器的值加到基址暫存器，或從中減去。如果傳遞了某個條件，則將值寫回基址暫存器。shift 指定五種移位型別之一。請參閱載入/儲存字或無符號位元組：調整的暫存器事先變址 (AM 2)。

 [ basereg ], #+/- value

事後變址選項：
將 12 位立即數偏移量的值加到基址暫存器，或從中減去。然後將值寫回基址暫存器。請參閱載入/儲存字或無符號位元組：立即數事後變址 (AM 2)。

 [ basereg ], +/- indexreg

事後變址選項：
如果傳遞某個條件，則將變址暫存器的值加到基址暫存器，或從中減去。然後將值寫回基址暫存器。請參閱載入/儲存字或無符號位元組：暫存器事後變址 (AM 2)。

 [ basereg ], +/- indexreg , shift # sh_imm

事後變址選項：
如果傳遞某個條件，則將移位的或循環移位的變址暫存器值加到基址暫存器，或從中減去。然後將值寫回基址暫存器。shift 指定五種移位型別之一。請參閱載入/儲存字或無符號位元組：調整的暫存器事後變址 (AM 2)。

上述九種子模式全都可用於指令 LDR、LDRB、STR 及 STRB。對於指令 LDRBT、LDRT、STRBT 及 STRBT，僅可使用事後變址選項。對於指令 PLD，僅可使用偏移量選項。

其中：

 P

P == 0 表示事後變址定址。基址暫存器的值用作記憶體地址。偏移量隨後應用於基址暫存器值，並寫回基址暫存器。
P == 1 表示偏移量定址或事先變址定址。（這由 W 位確定。）記憶體地址是通過將偏移量應用於基址暫存器值而產生的。

 U

表示是偏移量加到基址還是從中減去：
如果 U == 1，則偏移量加到基址上。
如果 U == 0，則偏移量從基址中減去。

 B

表示無符號位元組或字訪問：
B == 1 表示無符號位元組。
B == 0 表示字訪問。

 L

表示“載入”或“儲存”：
L==1 表示“載入”。
L == 0 表示“儲存”。

 

W 位的含義取決於 P 的設定：

 

 P==0

 P==1

 W==0

指令是 LDR、LDRB、STR 或 STRB。執行普通記憶體訪問。

基址暫存器不更新（偏移量定址）。

 W==1

指令是 LDRBT、LDRT、STRBT 或 STRT。執行非特權記憶體訪問。

計算的記憶體地址寫回基址暫存器（事先變址定址）。
