加载与存储字或无符号字节概述

“加载与存储字或无符号字节”寻址模式分成 9 个子模式。它们用于计算加载与存储无符号字节指令的地址。此寻址模式的一般语法在后续部分提供。

一般语法

LDR|STR [condition] [B] [T] dest, addr_mode

其中：

 LDR

指定“加载寄存器”指令。

 STR

指定“存储寄存器”指令。

 condition

16 个条件之一。请参阅条件代码状态。

 B

将指令扩展到“加载寄存器字节”(LDRB) 指令或“存储寄存器字节”( STRB ) 指令。如需有关详细信息,请参阅 LDRB 指令或 STRB 指令。

 T

将指令扩展到“带转译加载寄存器”(LDRT) 指令，或“带转译存储寄存器”( STRT ) 指令。如需有关详细信息，请参阅 LDRT 指令或 STRT 指令。

 dest

指定目标寄存器。

 

addr_mode 是以下 9 种子模式之一：

子模式

说明

 [ basereg , #+/- value ]

偏移量选项：
将 12 位立即数偏移量的值加到基址寄存器，或从中减去。请参阅加载/存储字或无符号字节：立即数偏移量 (AM 2)。

 [ basereg , +/- indexreg ]

偏移量选项：
将变址寄存器的值加到基址寄存器，或从中减去。请参阅加载/存储字或无符号字节：寄存器偏移量 (AM 2)。

 [ basereg , +/- indexreg , shift # sh_imm ]

偏移量选项：
将移位的或循环移位的变址寄存器值加到基址寄存器，或从中减去。shift 指定五种移位类型之一。请参阅加载/存储字或无符号字节：调整的寄存器偏移量 (AM 2)。

 [ basereg #+/- value ]!

事先变址选项：
将 12 位立即数偏移量的值加到基址寄存器，或从中减去。如果传递了某个条件，则将值写回基址寄存器。请参阅加载/存储字或无符号字节：立即数事先变址 (AM 2)。

 [ basereg , +/- indexreg ]!

事先变址选项：
将变址寄存器的值加到基址寄存器，或从中减去。如果传递了某个条件，则将值写回基址寄存器。请参阅加载/存储文字或未有符号字节：寄存器预先变址 (AM 2)。

 [ basereg , +/- indexreg , shift # sh_imm ]!

事先变址选项：
将移位的或循环移位的变址寄存器的值加到基址寄存器，或从中减去。如果传递了某个条件，则将值写回基址寄存器。shift 指定五种移位类型之一。请参阅加载/存储字或无符号字节：调整的寄存器事先变址 (AM 2)。

 [ basereg ], #+/- value

事后变址选项：
将 12 位立即数偏移量的值加到基址寄存器，或从中减去。然后将值写回基址寄存器。请参阅加载/存储字或无符号字节：立即数事后变址 (AM 2)。

 [ basereg ], +/- indexreg

事后变址选项：
如果传递某个条件，则将变址寄存器的值加到基址寄存器，或从中减去。然后将值写回基址寄存器。请参阅加载/存储字或无符号字节：寄存器事后变址 (AM 2)。

 [ basereg ], +/- indexreg , shift # sh_imm

事后变址选项：
如果传递某个条件，则将移位的或循环移位的变址寄存器值加到基址寄存器，或从中减去。然后将值写回基址寄存器。shift 指定五种移位类型之一。请参阅加载/存储字或无符号字节：调整的寄存器事后变址 (AM 2)。

上述九种子模式全都可用于指令 LDR、LDRB、STR 及 STRB。对于指令 LDRBT、LDRT、STRBT 及 STRBT，仅可使用事后变址选项。对于指令 PLD，仅可使用偏移量选项。

其中：

 P

P == 0 表示事后变址寻址。基址寄存器的值用作内存地址。偏移量随后应用于基址寄存器值，并写回基址寄存器。
P == 1 表示偏移量寻址或事先变址寻址。（这由 W 位确定。）内存地址是通过将偏移量应用于基址寄存器值而生成的。

 U

表示是偏移量加到基址还是从中减去：
如果 U == 1，则偏移量加到基址上。
如果 U == 0，则偏移量从基址中减去。

 B

表示无符号字节或字访问：
B == 1 表示无符号字节。
B == 0 表示字访问。

 L

表示“加载”或“存储”：
L==1 表示“加载”。
L == 0 表示“存储”。

 

W 位的含义取决于 P 的设置：

 

 P==0

 P==1

 W==0

指令是 LDR、LDRB、STR 或 STRB。执行普通内存访问。

基址寄存器不更新（偏移量寻址）。

 W==1

指令是 LDRBT、LDRT、STRBT 或 STRT。执行非特权内存访问。

计算的内存地址写回基址寄存器（事先变址寻址）。
