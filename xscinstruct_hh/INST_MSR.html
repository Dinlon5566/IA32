MSR - 从 ARM* 核心寄存器移动到状态寄存器

语法

MSR[condition] CPSR_bit, #value
MSR[condition] CPSR_bit, source
MSR[condition] SPSR_bit, #value
MSR[condition] SPSR_bit, source

其中：

 condition

16 个条件之一。请参阅条件代码状态。

 bit

有效的掩码位：
c - 控制字段掩码位 [16]
x - 扩展字段掩码位 [17]
s - 状态字段掩码位 [18]
f - 标志字段掩码位 [19]

 value

8 位立即数
请参阅寻址模式 1：数据处理操作数

 source

源寄存器

说明

MSR 指令将 value 或是 source 中包含的值移动到 CPSR 或 SPSR。
MSR 指令可用于更新处理器模式、条件代码标志或启用中断。要选择 CPSR，位 [22] 需要设置为 0。要选择 SPSR，位 [22] 需要设置为 1。
condition 必须是有效值；否则将指令当作 NOP。

备注

SPSR 不应在“系统”或“用户”模式中访问。CSPR 的 T 位不得更改。

寻址模式

请参阅“寻址模式 1 - 数据处理操作数 - 立即数”
