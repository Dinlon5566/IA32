按立即数算术右移 (AM 1)

语法

op2, ASR #sh_imm

其中：

 op2

指定包含要移位的值的寄存器。

 ASR

指示算术右移。

 sh_imm

指定移位。此值介于 1 与 32 之间。移 32 位使用 sh_imm == 0 编码表示。

说明

数据处理操作数“按立即数算术右移”执行以下操作（伪代码说明）：

if sh_imm == 0 then
  if op2[31] == 0 then
      addr_mode = 0
      shifter_carry_out = op2[31]
  else /* op2[31] == 1 */
      addr_mode = 0xFFFFFFFF
      shifter_carry_out = op2[31]
else /* sh_imm > 0 */
  addr_mode = op2 Arithmetic_Shift_right sh_imm
  shifter_carry_out = op2[sh_imm -1]

此数据处理操作数提供右移寄存器（除以 2 的常数次幂）的有符号值。

此指令操作数代表按立即数算术右移的寄存器 op2 的值，该立即数介于 1 与 32 之间。在腾出的位中，将插入 op2 ( op2 [31] ) 的符号位。最后移出的位是移位器的进位。
