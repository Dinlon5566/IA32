ASR(1) (Thumb*) - 算术右移

语法

ASR dest, op1, #value

其中：

dest

目标寄存器

op1

包含第一个操作数的寄存器

value

5 位立即数，形成 [1...32] 范围内的移位。

说明

此格式 ASR 指令向右移位，op1 中的位按照 value 中指定的数量进行移位，并将结果存储到 dest。如果 value 包含的移位量在 [1...31] 范围内，则直接编码。如果 value 包含的移位量等于 [32]，则编码成 value ==0。结果表示将寄存器中的有符号值除以 2 的常数次幂。

寻址模式

无操作数。
