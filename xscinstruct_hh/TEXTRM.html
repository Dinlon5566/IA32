TEXTRM - 提取与传输

语法

TEXTRMsignlength[Condition] dest, op1, #Imm3

其中：

sign
S -- 符号扩展
U -- 无符号
length
所需的数据长度，可以由以下字符之一指定：
B -- 提取 8 位（字节）操作
H -- 提取 16 位（半字）操作
W -- 提取 32 位（字）操作
condition
可选。16 个条件之一。请参阅：条件执行。
dest
目标，“英特尔 XScale(R)”技术数据寄存器
op1
源操作数，“英特尔(R) 无线 MMX™”技术协处理器数据寄存器
Imm3
指定提取哪个字节/半字/字

说明

从“英特尔(R) 无线 MMX™”技术源寄存器 op1 提取 3 位立即数字段数据指定的 8-/16-/32 位矢量，并将它传输到指定的“英特尔 XScale(R)”核心寄存器 dest。
如果相应地设置了符号，则数据讲进行符号扩展。

指令中指定的任何条件必须与条件代码状态相匹配，否则不执行该指令。

备注

将 R15 指定为目标寄存器时，将会产生无法预知的结果。

操作

if (B Specified) then

dest[7:0] = op1[Byte #Imm[2:0]]
dest[31:8]= (S Specified) ?SignReplicate(op1[Byte #Imm3[2:0]],24): 0

else if (H Specified)

dest[15:0] = op1[Half #Imm[1:0]]
dest[31:16]=(S Specified) ?SignReplicate(op1[Half #Imm3[1:0]],16): 0

else if (W Specified) then

dest[31:0] = op1[Word #Imm3[0]]
