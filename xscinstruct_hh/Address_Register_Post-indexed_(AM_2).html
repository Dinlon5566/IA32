寄存器事后变址 (AM 2)

语法

[basereg], #+/-indexreg

其中：

 basereg

指定包含基址的寄存器。

 indexreg

指定包含要加到 basereg 或从中减去的偏移量的寄存器。

说明

寻址模式“寄存器事后变址”执行以下操作（伪代码说明）：

addr =basereg
if ConditionPassed (condition) then
  if U == 1 then
      basereg = basereg + indexreg
  else/* U == 0 */
      basereg = basereg - indexreg

此寻址模式使用基址寄存器 basereg 的值作为内存访问地址。

如果指令中指定的条件与条件代码状态匹配，则将变址寄存器 indexreg 的值加到 basereg 的值，或从中减去。然后它写回基址寄存器 basereg。

 indexreg

B 位表示无符号字节 ( B==1 ) 或字访问 ( B==0 )。

L 位表示“加载”( L==1 ) 或“存储”( L==0 ) 指令。

 备注

此寻址模式编码为按 LSL 调整寄存器偏移量，调整量为零。
指令 LDRBT、LDRT、STRBT 与 STRT 仅允许使用事后变址寻址模式。对于这些指令，会修改以上位模式：位 [21]（W 位）是 1，而不是如上所示的 0。

 注意

如果 R15 指定为寄存器 basereg 或 indexreg，则结果将会无法预知。
为 basereg 与 indexreg 指定相同的寄存器时，结果将会无法预知。

示例

LDR    R10, [R8], R11
