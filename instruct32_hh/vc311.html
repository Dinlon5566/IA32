SYSENTER - 快速转换到系统调用入口

操作码

指令

说明

0F 34

SYSENTER

转换到“系统调用入口”

说明

SYSENTER 指令是在奔腾(R) II 处理器上引入的“快速系统调用”功能的一部分。SYSENTER 指令进行过专门的优化，能够以最佳性能转换到保护环 0 (CPL 0)。

根据操作系统在特定的型号专用寄存器中指定的值，SYSENTER 指令设置以下寄存器。

CS 寄存器设置为 (SYSENTER_CS_MSR) 的值；EIP 寄存器设置为 (SYSENTER_EIP_MSR) 的值；SS 寄存器设置为（8 与 SYSENTER_CS_MSR 的值之和）；ESP 寄存器设置为 (SYSENTER_ESP_MSR) 的值

处理器不保存用户堆栈或返回地址信息，也不保存任何寄存器。

SYSENTER 与 SYSEXIT 指令不构成一对调用/返回；因此，用户代码执行的系统调用“存根”例程（通常在共享库或 DLL 中）必须负责保存所需的寄存器状态，以创建一对系统调用/返回。

SYSENTER 指令总是转换到 CPL 0 级的普通保护模式内核。SYSENTER 可以从除实模式以外的所有模式激活。此指令要求操作系统满足以下条件：

    目标环 0 代码段的 CS 选择器是 32 位，映射到普通的 0-4 GB 地址空间，并拥有执行与读取权限。目标环 0 堆栈段的 SS 选择器是 32 位，映射到普通 0-4 GB 地址空间，并拥有读取、写入及访问权限。此选择器（目标环 0 SS 选择器）被赋予新值（CS 选择器 + 8）。

操作系统通过在处理器中使用型号专用寄存器，给环 0 入口的 CS、EIP、SS 以及 ESP 提供值。这些寄存器可以使用 RDMSR 与 WRMSR 指令进行读取和写入。在支持此项功能的未来处理器中，这些寄存器地址定义为固定到以下地址。

 

名称

说明

地址

SYSENTER_CS_MSR

目标环 0 CS 选择器

174H

SYSENTER_ESP_MSR

目标环 0 ESP

175H

SYSENTER_EIP_MSR

目标环 0 入口 EIP

176H

此项功能的存在与否由 CPUID 的位 11“SYSENTER 存在”(SEP) 指出。检测到存在 SEP 位的操作系统还必须判断处理器的系列与型号是否符合，以确保确实存在 SYSENTER/SYSEXIT 指令。例如：

IF (CPUID SEP bit is set)
IF (Family == 6) AND (Model < 3) AND (Stepping < 3)
THEN
Fast System Call NOT supported
FI;
ELSE Fast System Call is supported
FI

奔腾 Pro 处理器（型号 1）返回一组 SEP CPUID 功能位，但不支持 SYSENTER/SYSEXIT 指令。

操作

SYSENTER
IF CR0.PE == 0 THEN #GP(0)
IF SYSENTER_CS_MSR == 0 THEN #GP(0)

EFLAGS.VM := 0 // Prevent VM86 mode
EFLAGS.IF := 0 // Mask interrupts

CS.SEL := SYSENTER_CS_MSR // Operating system provides CS

// Set rest of CS to a fixed value
CS.SEL.CPL := 0 // CPL 0
CS.SEL.BASE := 0 // Flat segment
CS.SEL.LIMIT := 0xFFFF // 4G limit
CS.SEL.G := 1 // 4 KB granularity
CS.SEL.S := 1
CS.SEL.TYPE_xCRA := 1011 // Execute + Read, Accessed
CS.SEL.D := 1 // 32 bit code
CS.SEL.DPL := 0
CS.SEL.RPL := 0
CS.SEL.P := 1
SS.SEL := CS.SEL+8

// Set rest of SS to a fixed value
SS.SEL.BASE := 0 // Flat segment
SS.SEL.LIMIT := 0xFFFF // 4G limit
SS.SEL.G := 1 // 4 KB granularity
SS.SEL.S := 1
SS.SEL.TYPE_xCRA := 0011 // Read/Write, Accessed
SS.SEL.D := 1 // 32 bit stack
SS.SEL.DPL := 0
SS.SEL.RPL := 0
SS.SEL.P := 1

ESP := SYSENTER_ESP_MSR
EIP := SYSENTER_EIP_MSR

异常

#GP(0) - 如果 SYSENTER_CS_MSR 包含零。

数值异常

无。

实地址模式异常

#GP(0) - 如果未启用保护模式。
