清除中断标志；清除中断标志时禁用中断

说明

清除 EFLAGS 寄存器中的 IF 标志。其它标志不受影响。清除 IF 标志时会导致处理器忽略可屏蔽的外部中断。IF 标志与 CLI 及 STI 指令不影响异常与 NMI 中断的生成。

影响的标志

如果 CPL 等于或小于 IOPL，则 IF 清除为 0；否则它不受影响。EFLAGS 寄存器中的其它标志不受影响。
