SHUFPS - 单精度浮点值乱序

操作码

指令

说明

0F C6 /r ib

SHUFPS xmm1, xmm2/m128, imm8

对 xmm1 与 xmm1/m128 中由 imm8 选择的压缩单精度浮点值执行乱序处理，结果放入 xmm1。

说明

对于目标操作数（第一个操作数）中的四个压缩单精度浮点值，将其中两个移到目标操作数的低位四字；对于源操作数中的四个压缩单精度浮点值，将其中两个移到目标操作数的高位四字（请参阅“图 3-17”）。选择操作数（第三个操作数）确定要将哪些值移到目标操作数。

图 3-17. SHUFPS 乱序操作

源操作数可以是 XXM 寄存器或 128 位内存位置。目标操作数是 XMM 寄存器。选择操作数是一个 8 位立即数：位 0 与 1 选择目标操作数中要移到结果的低位双字的值，位 2 与 3 选择目标操作数中要移到结果的第二个双字的值，位 4 与 5 选择目标操作数中要移到结果的第三个双字的值，位 6 与 7 选择目标操作数中要移到结果的高位双字的值。

保护模式异常

#GP(0) - CS、DS、ES、FS 或 GS 段中的内存操作数有效地址非法。如果内存操作数未对齐 16 字节边界，不论是哪一段。

#SS(0) - SS 段中的地址非法。

#PF(错误代码) - 页错误。

#NM - 如果 CR0 中的 TS 设置为 1。

#XM - 如果发生无掩码 SIMD 浮点异常，且 CR4 中的 OSXMMEXCPT 是 1。

#UD - 如果发生无掩码 SIMD 浮点异常，且 CR4 中的 OSXMMEXCPT 是 0。如果 CR0 中的 EM 设置为 1。如果 CR4 中的 OSFXSR 是 0。如果 CPUID 功能标志 SSE 是 0。
