LOOP/LOOPcc - 根据 ECX 计数器循环

操作码

指令

说明

E2 cb

LOOP rel8

递减计数；计数 0 时短跳转

E1 cb

LOOPE rel8

递减计数；计数 0 且 ZF=1 时短跳转

E1 cb

LOOPZ rel8

递减计数；计数 0 且 ZF=1 时短跳转

E0 cb

LOOPNE rel8

递减计数；计数 0 且 ZF=0 时短跳转

E0 cb

LOOPNZ rel8

递减计数；计数 0 且 ZF=0 时短跳转

说明

将 ECX 或 CX 寄存器用作计数器，执行循环操作。LOOP 指令每执行一次，计数寄存器递减，然后检查是否为 0。如果计数为 0，则终止循环，并继续执行程序中 LOOP 指令后面的指令。如果计数器不为零，则执行近跳转，跳转到目标（跳转目标）操作数，假定它是位于循环开头的指令。如果地址大小属性是 32 位，则将 ECX 寄存器用作计数寄存器；否则使用 CX 寄存器。

目标指令使用相对偏移量（相对于 EIP 寄存器中指令指针当前值的有符号偏移量）指定。汇编代码中此偏移量通常指定为标号，但是在机器代码级别，它编码为有符号 8 位立即数，并加到指令指针上。此指令可以使用 -128 到 +127 的偏移量。

某些形式的循环指令 (LOOPcc) 还可以将 ZF 标志用作循环终止条件，在计数变为零之前终止循环。在这些形式的指令中，条件代码 (cc) 与每条指令关联，用于指明测试条件。这里，LOOPcc 指令本身不影响 ZF 标志的状态；ZF 标志由循环中的其它指令更改。

操作

IF AddressSize 32
THEN
Count is ECX;
ELSE (* AddressSize 16 *)
Count is CX;
FI;
Count Count - 1;

IF instruction is not LOOP
THEN
IF (instruction LOOPE) OR (instruction LOOPZ)
THEN
IF (ZF =1) AND (Count 0)
THEN BranchCond 1;
ELSE BranchCond 0;
FI;
FI;
IF (instruction LOOPNE) OR (instruction LOOPNZ)
THEN
IF (ZF =0 ) AND (Count 0)
THEN BranchCond 1;
ELSE BranchCond 0;
FI;
FI;
ELSE (* instruction LOOP *)
IF (Count 0)
THEN BranchCond 1;
ELSE BranchCond 0;
FI;
FI;
IF BranchCond 1
THEN
EIP EIP + SignExtend(DEST);
IF OperandSize 16
THEN
EIP EIP AND 0000FFFFH;
FI;
ELSE
Terminate loop and continue program execution at EIP;
FI;
