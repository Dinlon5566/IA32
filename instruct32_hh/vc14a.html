ANDPS - 压缩单精度浮点值逻辑位与

操作码

指令

说明

0F 54 /r

ANDPS xmm1,xmm2/m128

对 xmm2/m128 同 xmm1 执行逻辑位“与”运算。

说明

在源操作数（第二个操作数）与目标操作数（第一个操作数）中的四个压缩单精度浮点值上执行逻辑位“与”，结果存储到目标操作数。源操作数可以是 XMM 寄存器或 128 位内存位置。目标操作数是 XMM 寄存器。

操作

DEST[127-0] DEST[127-0] BitwiseAND SRC[127-0];

英特尔(R) C++ 编译器等价内部函数

ANDPS __m128 _mm_and_ps(__m128 a, __m128 b)

SIMD 浮点异常

无。

保护模式异常

#GP(0) - CS、DS、ES、FS 或 GS 段中的内存操作数有效地址非法。如果内存操作数未对齐 16 字节边界，不论是哪一段。

#SS(0) - SS 段中的地址非法。

#PF(错误代码) - 页错误。

#NM - 如果 CR0 中的 TS 设置为 1。

#XM - 如果发生无掩码 SIMD 浮点异常，且 CR4 中的 OSXMMEXCPT 是 1。

#UD - 如果发生无掩码 SIMD 浮点异常，且 CR4 中的 OSXMMEXCPT 是 0。如果 CR0 中的 EM 设置为 1。如果 CR4 中的 OSFXSR 是 0。如果 CPUID 功能标志 SSE 是 0。

#GP(0) - 如果内存操作数未对齐 16 字节边界，不论是哪一段。中断 13 - 如果操作数的任何部分出现在从 0 到 0FFFFH 的有效地址空间之外。

#NM - 如果 CR0 中的 TS 设置为 1。

#XM - 如果发生无掩码 SIMD 浮点异常，且 CR4 中的 OSXMMEXCPT 是 1。
