MOVDDUP - 移动一个双精度浮点值并复制

 

指令

说明

 

MOVDDUP xmm1, xmm2/m64

将 XMM2/Mem 中代表低位 DP 数据元素的 64 位移到 XMM1 寄存器并复制。

说明

线性地址对应于引用的内存数据的最低有效字节。指出内存地址时，将加载内存位置 m64 处 8 个字节的数据。使用寄存器-寄存器形式的此操作时，将复制 128 位源寄存器的下半部，并复制到 128 位目标寄存器。

操作

if (source == m64) {

// load instruction

xmm1[63-0] = m64;

xmm1[127-64] = m64;

}

else {

// move instruction

xmm1[63-0] = xmm2[63-0];

xmm1[127-64] = xmm2[63-0];

}

异常

无
