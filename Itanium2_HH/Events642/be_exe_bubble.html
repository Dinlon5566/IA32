BE_EXE_BUBBLE

 
标题：主管道中执行单元暂停导致的全管道气泡
类别：暂停事件
IAR/DAR/OPC：N/N/N
事件代码：0x02
最大每周期增量：1

定义：统计主管道中“执行单元”引起的暂停导致的全管道气泡数。

备注

此事件的各个产生原因之间不排优先级，因为不需要这么做（各个产生原因之间彼此独立，其中有一些是同时触发的，因此都应该加以统计）。

扩展

PMC.umask

说明

ALL

b0000

后端被 exe 暂停

GRALL

b0001

由于 GR/GR 或 GR/加载相关，后端被 exe 暂停

FRALL

b0010

由于 FR/FR 或 FR/加载相关，后端被 exe 暂停

PR

b0011

由于 PR 相关，后端被 exe 暂停

ARCR

b0100

由于 AR 或 CR 相关，后端被 exe 暂停

GRGR

b0101

由于 GR/GR 相关，后端被 exe 暂停

CANCEL

b0110

由于取消加载，后端被 exe 暂停

BANK_SWITCH

b0111

由于寄存器库切换，后端被 exe 暂停

ARCR_PR_CANCEL_BANK

b1000

ARCR、PR、CANCEL 或 BANK_SWITCH
