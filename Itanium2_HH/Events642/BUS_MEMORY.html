BUS_MEMORY

标题：总线内存事务
类别：前端总线事件
IAR/DAR/OPC：N/N/N
事件代码：0x8A
最大每周期增量：1

定义：统计总线内存事务（如内存读取失效、保留内存读取、内存读取以及内存写入事务）数。

 

扩展

PMC.umask

说明

---

b00xx

（*不统计*）

---

b0100

（*不统计*）

EQ_128BYTE.IO

b0101

非 CPU 优先级代理进行的完全缓存线事务（BRL、BRIL、BWL）数

EQ_128BYTE.SELF

b0110

本地处理器进行的完全缓存线事务（BRL、BRIL、BWL）数

EQ_128BYTE.ANY

b0111

CPU 或非 CPU 进行的完全缓存线事务（BRL、BRIL、BWL）的数量（所有事务）。
