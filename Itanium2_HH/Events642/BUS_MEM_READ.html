BUS_MEM_READ

标题：完全缓存线 D/I 内存读取、读取失效以及 BRIL
类别：前端总线事件
IAR/DAR/OPC：N/N/N
事件代码：0x8B
最大每周期增量：1

定义：统计完全缓存线（128 字节）数据/代码内存读取 (BRL)、完全缓存线内存读取失效 (BRIL) 以及 0 字节内存读取失效 (BIL) 事务数。

 

扩展

PMC.umask

说明

---

b0000

（*不统计*）

BIL.IO

b0001

非 CPU 优先级代理进行的 BIL 0 字节内存读取失效事务数

BIL.SELF

b0010

本地处理器进行的 BIL 0 字节内存读取失效事务数

BIL.ANY

b0011

CPU 或非 CPU 进行的 BIL 0 字节内存读取失效事务数（所有事务）

---

b0100

（*不统计*）

BRL.IO

b0101

非 CPU 优先级代理进行的完全缓存线内存读取事务数

BRL.SELF

b0110

本地处理器进行的完全缓存线内存读取事务数

BRL.ANY

b0111

CPU 或非 CPU 进行的完全缓存线内存读取事务数（所有事务）。

---

b1000

（*不统计*）

BRIL.IO

b1001

非 CPU 优先级代理进行的完全缓存线内存读取失效事务数

BRIL.SELF

b1010

本地处理器进行的完全缓存线内存读取失效事务数

BRIL.ANY

b1011

CPU 或非 CPU 进行的完全缓存线内存读取失效事务数（所有事务）。

---

（*不统计*）

ALL.IO

本地处理器进行的所有内存读取事务

ALL.ANY

b1111
