BR_MISPRED_DETAIL

标题：FE 分支预测失误细节
类别：分支事件
IAR/DAR/OPC：Y/N/Y
事件代码：0x5B
最大每周期增量：3

定义：统计失效的分支数。PMC.umask 的所有 16 种值都有效，这是为了能够根据预测结果（前端预测失误的路径或目标地址）与分支类型提供信息

 

扩展

PMC.umask

说明

ALL.ALL_PRED

b0000

所有分支类型，不论预测结果如何

ALL.CORRECT_PRED

b0001

所有分支类型，正确预测的分支（结果与目标）

ALL.WRONG_PATH

b0010

所有分支类型，分支方向错误导致预测失误的分支

ALL.WRONG_TARGET

b0011

所有分支类型，执行分支的目标错误导致预测失误的分支

IPREL.ALL_PRED

b0100

仅限 IP 相对分支，不论预测结果如何

IPREL.CORRECT_PRED

b0101

仅限 IP 相对分支，正确预测的分支（结果与目标）

IPREL.WRONG_PATH

b0110

仅限 IP 相对分支，分支方向错误导致预测失误的分支

IPREL.WRONG_TARGET

b0111

仅限 IP 相对分支，执行分支的目标错误导致预测失误的分支

RETURN.ALL_PRED

b1000

仅限返回型分支，不论预测结果如何

RETURN.CORRECT_PRED

b1001

仅限返回型分支，正确预测的分支（结果与目标）

RETURN.WRONG_PATH

b1010

仅限返回型分支，分支方向错误导致预测失误的分支

RETURN.WRONG_TARGET

b1011

仅限返回型分支，执行分支目标错误导致预测失误的分支

NTRETIND.ALL_PRED

b1100

仅限非返回型间接分支，不论预测结果如何

NTRETIND.CORRECT_PRED

b1101

仅限非返回型间接分支，正确预测的分支（结果与目标）

NTRETIND.WRONG_PATH

b1110

仅限非返回型间接分支，分支方向错误导致预测失误的分支

NTRETIND.WRONG_TARGET

b1111

仅限非返回型间接分支，执行分支目标错误导致预测失误的分支
