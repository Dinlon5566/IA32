前一條 PV 或 NP 指令

Prev_PV/NP 表示一種配對限制衝突。

為其發出 Prev_PV/NP 的指令無法配對，原因在於前一條發到 U 管道的指令是 PV 指令或 NP 指令。

建議

調整程式碼結構，使得 PV 指令是指令對中的第二條指令。

示例：PV 指令

在本例中，JNZ 發到 U 管道。不過，因為 JNZ 是 PV 指令，它無法同下一條指令配對。也就是說，即使 ADD 是 UV 指令，這兩條指令也無法配對。


1. jnz label5
label5:
2. add eax,43534

示例：NP 指令

在本例中，ADD 是 UV 指令。不過，由於 NEG 是 NP 指令，因此這兩條指令無法配對。


add eax,43534
neg
